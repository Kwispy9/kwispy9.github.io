Using this whiteboard is very simple for both the client and the server host.

IMPORTANT: You are going to need python installed in order to host a server.

Hosting a Server

To host a server, first open your command prompt in the file location that has the server.py file and run the following command:

python server.py

After that, the server should be running. It will give you a list of IPs, saying which IP the server started at.

If you want to start the server with a different port than 5555, just open the server.py file in any text editor and locate server_port = 5555 from the screenshot below.

Change 5555 to whatever you like, and you're good to go. Please note that valid ports are only numbers between 1 and 65535.

NOTE: The command prompt where you ran python server.py must stay open to keep the server running.

Joining a Server

To join a server, run the client.exe file, input the host's server IP and Port (the default port is 5555), and click OK.

NOTE: The current version of this app requires the inputted IP to be the actual IP; you can't input a domain as an alias to the IP.

I will be working on making the ability to use an alias at a different time.

Troubleshooting

If you're having trouble with anything, this section will cover the most likely causes to your troubles.

If you're having trouble connecting to a server, make sure you've enetered the correct server IP and Port. If that's not the issue, the most likely cause is that the server you're trying to connect to is not on your local network (LAN). If you want to connect to a server outside of your local netowrk (LAN), then make sure said server is port-forwarded.

If you're still having issues, you can look up your issue relating to servers, and connecting to servers. Or, the IP you've entered is still incorrect, in that case, the correct IP may be your router's IP address.
